# Git

## Git New Branch

```sh
git branch dev-ilham
```

## Git Show remote

```sh
git remote -v
```

## Git Switch Branch

```sh
git checkout (name branch)
```

## Git Push Specific Remote or Branch

```sh
git push -u (name remote) (name branch)
```

> *example*
>  
>```sh
>git push -u origin dev
>```

## Git tag

## Git Change Commit

```sh
git commit --amend 
```

## Combine Multiple Git Commits into One

```sh
git rebase -i HEAD~
```
